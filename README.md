# 🔥 Fire Detection System

A simple and effective real-time fire detection system using YOLOv8 and OpenCV.
It monitors live video, detects fire instantly, triggers an alarm, and sends a WhatsApp alert using Twilio.

## 🚀 Features

* Real-time fire detection
* Instant WhatsApp alerts 📱
* Alarm trigger on detection 🔊
* Saves detected frames automatically

## 🛠 Tech Stack

Python • OpenCV • YOLOv8 • Twilio API • Pygame

## ▶️ Run the Project

```bash
python firedetection.py
```
